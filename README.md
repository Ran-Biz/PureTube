# PureTube

**Tracker-Free YouTube Search** — Watch YouTube without being watched.

Search and enjoy YouTube videos without trackers, algorithms, or personalized recommendations. PureTube gives you a pure, distraction-free video watching experience.

## Features

- 🛡️ **No Trackers** — Uses `youtube-nocookie.com` embeds for privacy
- 🧠 **No Algorithm** — No recommendations or suggested videos
- 🔍 **Search Only** — Find exactly what you want, nothing more
- ⚡ **Fast & Lightweight** — Built with Bun + React

## Getting Started

### Prerequisites

- [Bun](https://bun.sh) runtime installed

### Installation

```bash
bun install
```

### Development

```bash
bun run dev
```

### Production

```bash
bun start
```

## Environment Variables

Copy `.env.example` to `.env` and configure your YouTube API key:

```
YOUTUBE_API_KEY=your_api_key_here
```
